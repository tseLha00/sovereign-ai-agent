# Sovereign AI Agent Demo

On-prem sovereign AI chatbot demo based on the Apertus LLM, exposing an OpenAI-compatible API surface and a web UI.
Includes QA (tests + performance baseline) and project documentation for traceable evidence.

## Repo structure
- `backend/` FastAPI backend exposing OpenAI-style endpoints
- `frontend/` Web UI for chat
- `docs/` Documentation (scope, architecture, decisions, QA, journal)
- `evidence/` Stored outputs (perf JSON, screenshots, meeting notes)
